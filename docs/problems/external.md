# External Problems

While EngiBench provides and maintains several problems, there are also notable external problems that follow the EngiBench API. These are **not maintained** by the EngiBench team and therefore are not guaranteed to work out of the box or be updated regularly.

*If you'd like to contribute an environment, please submit a PR by editing this [file](https://github.com/IDEALLab/EngiBench/blob/main/docs/problems/external.md). Additional instructions can be found inside that file.*
